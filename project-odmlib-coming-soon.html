Project: odmlib

The odmlib package enables Python developers to work with CDISC ODM data exchange standard and its extensions, such as Define-XML, as an object model.

odmlib Python Package

  • Industry: Clinical Research
  • Traget Audience: Developers working with ODM
  • Website: https://swhume.github.io
More details on this project will be posted soon, including a link to the repository.

Project Requirements

  • Python package that makes it easy for programmers to work with ODM in an object-oriented fashion.
  • Make it easy for Python programmers to create and load ODM files.
  • Make it easy to support standard ODM extensions like Define-XML.
  • Make it easy to create and work with custom ODM extensions.
